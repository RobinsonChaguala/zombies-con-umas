# zombies con umas

* Este es un videojuego desarrollado en Unity el cual simula y toma como inspiración el videojuego  Call of Duty: Black Ops zombies
* Su modo de juego principal es un modo cooperativo con amigos encargados de eliminar zombies con armas de fuego y un sistema de rondas identico al Black Ops zombies
* A futuro tengo planeado desarrollar más modos de juego manteniendo la esencia del modo cooperativo, además de incluir funcionalidades y la opción de modificar ciertos aspectos del juego para hacerlo más personalizable, como cambiar las skins tanto de los jugadores como de los zombies y demás cosas que se me vayan ocurriendo o me vayan recomendando

## Tecnologías usadas
### Motor: Unity
### Lenguaje: C#
### Librerias: Mirror, Unity
### Editor de codigo: Visual Studio Code
### Plataforma objetivo: PC
